本项目是基于easyspider对接第三方阅卷平台智学网实现的大模型AI无人阅卷。
大模型使用了kimi的api，注册有15元的免费额度，可以用于测试。
核心文件有3个： 389.json，main.py，pic2base64.py，其中后两个属于嵌入式脚本。
389.json属于流程文件，需放在tasks目录下。

有编程基础的把easyspider教程看完，就知道怎么用了。
教程地址：https://github.com/NaiboWang/EasySpider/wiki
